# SmartSenseFeed



This Project will provide following things :

- Feed Text
- Feed Image with Staggered View
- Feed AutoPlay Videos
- Plays videos inside RecyclerView cell.
- Precaching of videos(no loading of videos)
- Uses remote video URLs
